Question Asker

Try it out

Efficiently answer yes or no questions about a list of stuff.

List Input Screen

The initial screen for the tool. The description of purpose and area for action is presented clearly.

Question Input Screen

Where the user specifies the yes or no question. Clicking the button or hitting enter will advance to the next screen.

Question Answering Screen

The user is shown the question, the item currently the subject of the question, the actions used for answering the question, and the instructions for keyboard hotkeys. Undo will reverse the previous answer, allowing the user to quickly answer the question and go back if they think their answer was incorrect. The page is streamlined to make answering the question as fast as possible.

Results

The results screen shows the list of items split into the yes or no categories. The lists are in text areas for easy copying by the user.
