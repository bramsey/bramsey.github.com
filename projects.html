Minify List

Helps inventory your stuff and list it for sale online.

Landing page

A minimalist landing page that includes the value proposition and the basic actions.

Item Dashboard

The main area for organizing and modifying items. Items are presented in a sparse grid to allow easy scanning without overloading the user.

Filtering

The item dashboard provides filtering by tags. Items quickly blur in and out based on the selected filter.

Item Editing

When the user clicks an item in the dashboard, it changes switches into editing mode where the item name, tags, and description can be easily set. Tab and shift-tab may be used to switch between the fields and the items so the user doesn't need to use the mouse to enter all the information about their items.

Item Thumbnail

A red X appears when the user hovers over an item, giving the option to delete that item.

Item Selection

The mouse may be used to select a group of items at once. The user may also select multiple items by cmd/ctrl clicking individual items, or shift-clicking to select a range of items.

Group Actions

Once multiple items are selected, the action menu allows the user to delete or tag the whole group of items.

Public Items Page

Each user has a page for displaying their items to the public. Items with a price specified will show a price tag. The filtering works identically to the item dashboard filtering. In the user's account settings, they have multiple options for the display size of items on their public page. Larger item displays would better suit a smaller collection, while smaller displays would suit a larger collection.

Item Page

Each item has a page to display all of its detailed information. If the item has a price set, the price is shown and a paypal button is generated based on the user's email address or otherwise specified paypal address.

Lock Screen Maker

Try it out

Quickly save messages to the iPhone lock-screen by generating a perfectly sized image from inputted text.

Main Screen

The first screen the user sees upon visiting the site. It is formatted as a mobile web app, so it hides the browser bar and is suitable for saving as an app in iOS.

Text Input

When the user enters some text, an unopened color-selector appears. The default color for final text is black.

Color Selection

Upon clicking the color selector, the various color options slide out to the right. The selected color receives an outline.

Screen Image

After hitting the show button, the screen changes to display the given text as large as possible with the specified color such that it will fit perfectly in between the bars on an iPhone lock screen. From here the user needs to take a screen shot, then set the screen shot as their lock screen from the camera roll.

Prioritizer

Try it out

Quickly prioritize a list of stuff. Infers ordering based on prior input so you never have to compare two things twice.

List Input Screen

The initial screen for the tool. The description of purpose and area for action is presented clearly.

Ranking Screen

The whole page where ranking takes place.

Ranking Widget

The two items being compared are presented here. The buttons to perform the comparison are clearly visible, and instructions for hotkeys are provided so the mouse is not needed to compare items. After an action, an undo button appears that allows the user to reverse the last comparison, returning the tool to its previous state.

Ranking Lists

The lists of unranked and ranked items updates as the user makes each comparison. Not every comparison will result in a final ranking. Undo affects the ranking lists as well.

Results

After all items have been ranked, the ordered list appears in a text area for easy copying by the user.

Question Asker

Try it out

Efficiently answer yes or no questions about a list of stuff.

List Input Screen

The initial screen for the tool. The description of purpose and area for action is presented clearly.

Question Input Screen

Where the user specifies the yes or no question. Clicking the button or hitting enter will advance to the next screen.

Question Answering Screen

The user is shown the question, the item currently the subject of the question, the actions used for answering the question, and the instructions for keyboard hotkeys. Undo will reverse the previous answer, allowing the user to quickly answer the question and go back if they think their answer was incorrect. The page is streamlined to make answering the question as fast as possible.

Results

The results screen shows the list of items split into the yes or no categories. The lists are in text areas for easy copying by the user.
